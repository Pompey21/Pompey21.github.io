Welcome to Code Golf! The aim of this challenge is to solve the problem bellow in as few characters as possible using Haskell.

Given data type Stream represents the infinite sequence of repeated characters such as: a, a, a, a, a, a, a, b, c, d, e, e, e, e, e, b, b, b ...
Suppose we are able to encode the infinite sequence into a sequence of pairs (x,n) where n is the number of consequtive repetitions of x. The above example is encoded with: (a, 7), (b, 1), (c, 1), (d, 1), (e, 5), (b, 4), ... ,
Write a decode method that given an encoded sequence of a stream it returns a stream - initial flow.

data Stream a = Cons (a, Stream a)

decode :: Stream (a, Int) -> Stream a
